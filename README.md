# URL Shortener 🚀

A full-stack URL shortener built with FastAPI, PostgreSQL, and React.

## Features
- Shorten long URLs
- Custom short codes
- Redirect system
- Click tracking
- Optional user authentication

## Tech Stack
- Backend: FastAPI, SQLAlchemy, PostgreSQL
- Frontend: React
- Others: JWT Auth, REST API

## Status
🚧 Currently under development (adding signup,login system with analytics & dashboard)
